# Portifólio **tiagonl.dev.br**

```md

Projeto de portfólio pessoal desenvolvido para apresentar projetos, habilidades e experiências de forma moderna e responsiva.

## ✨ Visão geral

Este projeto é um site de portfólio construído com foco em performance, design limpo e boa experiência do usuário. Ele serve como vitrine principal dos meus projetos e tecnologias.

## 🚀 Tecnologias

- Next.js
- React
- HTML
- CSS

## 📦 Funcionalidades

- Página inicial com apresentação pessoal
- Seção de projetos
- Layout responsivo
- Interface moderna e otimizada

```

## ⚙️ Como rodar o projeto

```bash
npm install
npm run dev
```
