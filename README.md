# CommandBat Music

Sistema moderno de streaming de músicas com playlists, realtime e integração com Discord.

![Banner](https://tiagonl.com.br/ccf.jpg)

---

## Sobre o projeto

O CommandBat Music é uma plataforma focada em:

- streaming realtime
- integração com bots
- playlists públicas
- visual moderno
- websocket em tempo real

---

## Tecnologias

- Next.js
- Node.js
- Socket.IO
- MongoDB
- TailwindCSS

---

## Funcionalidades

- Login com Discord
- Player realtime
- Sistema de favoritos
- Histórico
- API própria
- Tema dark mode
